# Simple Mental Health Support Bot

## Project Overview
This project implements a **mental health chatbot** that can understand user input, assess risk levels for conditions like depression, anxiety, and suicidal thoughts, and provide appropriate responses. It combines **machine learning**, **deep learning**, **keyword detection**, and **sentiment analysis** to provide a safe and interactive mental health support tool.

## Features
- **Text Classification:** Uses Linear SVM and Bidirectional GRU models to classify messages into mental health categories.
- **Keyword Detection:** Identifies high-risk keywords related to suicide, self-harm, and depression.
- **Sentiment Analysis:** Uses VADER to assess the emotional tone of user messages.
- **Risk Scoring:** Combines model prediction, keywords, and sentiment to assign risk levels (low, medium, high).
